# Maruti Overseas Website

Welcome to the official repository for the Maruti Overseas website. This platform serves as a digital gateway for students aspiring to study abroad, providing resources, tools, and consultation services.

## 🚀 Project Overview

Maruti Overseas is a study abroad consultancy agency. This website allows students to:
- Explore study destinations and universities.
- Browse available courses.
- Use tools like the **GPA Converter**.
- Read insightful blogs and success stories.
- Book consultations with experts.

## 🛠️ Tech Stack

This project is built using modern web technologies to ensure performance, scalability, and a great user experience:

- **Frontend Framework**: [React](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **Forms**: [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)

## 📥 Installation & Setup Guide

Follow these steps to get the project up and running on your local machine.

### Prerequisites
Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Steps

1.  **Clone or Download the Repository**
    If you haven't already, download the project folder `Maruti Overseas(Kimi)` to your local machine.

2.  **Navigate to the App Directory**
    Open your terminal or command prompt in the project root folder and run:
    ```bash
    cd app
    ```

3.  **Install Dependencies**
    Run the following command to install all necessary libraries and packages:
    ```bash
    npm install
    ```

4.  **Run the Development Server**
    Start the local development server to view the website:
    ```bash
    npm run dev
    ```

5.  **Open in Browser**
    Once the server is running, you will see a local URL (usually `http://localhost:5173`). Open this link in your web browser.

### Building for Production

To create a production-ready build of the application, run:
```bash
npm run build
```
This will compile the project into the `dist` folder, optimized for deployment.

## 📂 Project Structure

- **`app/src`**: Contains the main source code.
  - **`components`**: Reusable UI components (Navbar, Footer, Sections, etc.).
  - **`pages`**: Individual page components (Home, About, Contact, Tools, etc.).
  - **`images`**: Static assets and images.
- **`app/public`**: Public assets.
